# Snake
 
O nome ***snake***, traduzido do inglês para o português, significa **cobra**. Apesar do nome ***Python*** não se referir diretamente à espécie de serpente, achei que seria interessante nomear o repositório fazendo uma referência a estes répteis.

Neste repositório, estão armazenados alguns exercícios que realizei durante meus estudos sobre Python. Existem **duas** pastas: uma contendo atividades consideradas mais **simples** e outra contendo atividades que, em minha opinião, foram mais complexas de se fazer.
